# 📱 WhatsApp Web Clone – Full Chat Application

**📌 Description**  
A sleek, fully functional, mobile-first web replica of **WhatsApp** (com.whatsapp).  
Real-time chat interface, send messages, voice notes, emojis, photos, make calls, manage multiple chats and profile — exactly like the real WhatsApp Web.

**✨ Features**  
- 📋 Beautiful chat list with unread badges & last messages  
- 💬 Fully interactive chat window with sent/received bubbles  
- 🔴 Online / last seen status indicators  
- ❤️ Real-time message sending with auto-reply simulation  
- 🎤 Voice note recording demo  
- 😃 Emoji picker with instant insertion  
- 📎 File / photo attachment simulation  
- 📞 Voice & video call buttons  
- 📱 100% responsive – collapses perfectly on mobile phones  
- 🌐 Works on desktop, tablet & mobile  
- 🔒 All data stored locally in browser (no real server)  

**🏷️ Tags / Keywords**  
`whatsapp-web`, `whatsapp-clone`, `web-whatsapp`, `chat-app-web`, `instant-messaging`, `whatsapp-web-demo`, `html5-chat`, `responsive-chat`, `whatsapp-web-clone`, `browser-messaging`

**🛠️ How to run**  
1. Download all 4 files into the **same folder**  
2. Open `index.html` in any modern browser  
3. Start chatting instantly — no login required!
